Search courses via /api/search with powerful filters and pagination.

Supported Query Parameters

  • q: Search keyword (title or description)
  • category: e.g., Math, Science
  • type: COURSE, CLUB, ONE_TIME
  • minAge / maxAge
  • minPrice / maxPrice
  • nextSessionDate: ISO-8601 (e.g., 2025-09-01T10:00:00Z)
  • sort: upcoming | priceAsc | priceDesc
  • page / size: Pagination support

Example:
/api/search?q=science&category=Science&minAge=8&sort=priceAsc&page=0&size=5
