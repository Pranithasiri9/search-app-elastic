Welcome to Course Search API

Use /api/search to search for courses.

You can also try these parameters:

  • category: e.g., Math, Science
  • type: COURSE, CLUB, ONE_TIME
  • minAge / maxAge
  • minPrice / maxPrice
  • startDate: ISO-8601 (e.g., 2025-09-01T10:00:00Z)
  • sort: upcoming, priceAsc, priceDesc
